# Dolphjs documentation

This guide will walk you through the installation process for setting up the DoplhJS documentation build using VitePress. DoplhJS is a JavaScript library/framework, and VitePress is a static site generator powered by Vue.js.

## Prerequisites

Before you begin, ensure that you have the following prerequisites installed on your system:

- Git
- Node: any 14.x version starting with v14.0.0 or greater
- Package Manager Yarn, Npm or Pnpm(https://yarnpkg.com/lang/en/docs/install/)
- A fork of the repo (for any contributions)

## Installation

To get started, simply clone the repository and install the dependencies.

```bash
# Clone the Repository
git clone https://github.com/dolphjs/docs

cd docs

# Install dependencies with your package manager
npm install
# or
yarn
# or
pnpm install

```

Open your web browser and visit http://localhost:5173/ to access the DoplhJS documentation.
